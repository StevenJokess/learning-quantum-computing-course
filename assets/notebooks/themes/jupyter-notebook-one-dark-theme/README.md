# Jupyter Notebook OneDark Theme

Already tested in Google Chrome and Apple Safari

### Installation

```bash
cp custom.css ~/.jupyter/custom/custom.css
pip install jupyterthemes
cp startup.py ~/.ipython/profile_default/startup/startup.py
```
